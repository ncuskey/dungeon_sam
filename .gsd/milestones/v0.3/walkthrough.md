# ALL MILESTONE WALKTHROUGH

## v0.3: Illumination & Touch (Feb 2026)
**Live:** [https://dungeonsam.site](https://dungeonsam.site)

### Features
- **Dynamic Lighting**: 
  - Dungeon is now dark by default.
  - Torches (orange point lights) appear randomly (10% chance) in hallways.
  - Player spawns with a guaranteed light source nearby.
- **Mobile Controls**:
  - Touch-friendly Start Screen (Tap to Start).
  - Virtual D-Pad (Left) for movement.
  - Action Buttons (Right) for Attack, Switch Weapon, Interact.
  - Disabled browser zoom/scroll for native app feel.
